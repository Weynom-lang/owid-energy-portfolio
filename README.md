# OWID Energy Portfolio

Python-based energy data analysis portfolio project using Our World in Data (OWID) datasets.

## Project Overview
This project explores energy trends across OECD, Non-OECD, Canada, and the world using Python, pandas, and matplotlib.

## Questions Covered
- Q1: OECD vs Non-OECD primary energy demand
- Q2: Fossil vs low-carbon energy consumption trends
- Q3: Carbon intensity of electricity
- Q4: Electricity mix shares and convergence/divergence
- Q5: Canada vs World primary energy mix
- Q6: Canada vs US, UK, and Germany comparison
- Q7: Growth drivers and percentage-point changes
- Q8: Policy and infrastructure implications

## Tools Used
- Python
- pandas
- matplotlib
