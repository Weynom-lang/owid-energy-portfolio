# Data Sources

## Primary Source
This project primarily uses energy datasets from **Our World in Data (OWID)**.

## Datasets Used
- Primary energy consumption and primary energy shares
- Energy per capita
- Electricity generation mix shares
- Carbon intensity of electricity
- CO2 emissions per capita

## Notes
- Comparisons were restricted to time periods with more stable coverage where necessary.
- Some figures include notes where visible breaks or shifts likely reflect dataset coverage/reporting changes rather than true one-year structural jumps.
- CO2 per capita comparisons were merged into the broader project to support cross-country analysis.

## Project Use
These datasets were used to analyze:
- OECD vs Non-OECD energy demand and per-capita trends
- Fossil vs low-carbon electricity transitions
- Canada vs World primary energy comparisons
- Canada vs peer-country comparisons
- Growth signals for likely future energy sources
